## ✨ 核心精進功能介紹

為了更好地輔助行者在澳洲夏季或日常繁忙生活中攝心專注，系統進行了深度重構，推出以下核心優化功能：

### 1. 🎙️ 現場法音錄製與「永久儲存」
* **一次錄音，終身使用**：行者只需親自錄製一句自己的準提咒音，系統會自動將音訊安全儲存在瀏覽器的本地空間（LocalStorage）中。
* **刷新不丟失**：下次打開網頁、刷新頁面甚至重啟電腦，原本錄好的法音都會自動載入，無需每次重新錄音，真正做到隨時隨地，提筆（滑鼠）即修。

### 2. ⚡ 預設「0.1 秒極速緊湊」間隔
* **專為熟練行者設計**：當咒語唸誦純熟後，最常用且最能防散亂的便是緊湊的連播。網頁現已將預設持誦間隔調整為最短的 **0.1 秒**，讓法音與心念一氣呵成，緊緊攝住心神。

### 3. 🔔 開咒偈與迴向「鳴磬儀式感」
* **開咒引磬**：在第一部分【開咒偈】下方特別增設了 `🔔 鳴磬一聲` 按鈕。正式功課前點擊，會模擬出極具穿透力與迴音的清脆引磬聲，提示身心放下萬緣，準備進入正課。
* **功德迴向**：在文末提供五種經典的準提功德迴向（準提鏡、消業、學業、人緣、財運），同樣配備鳴磬迴向按鈕，圓滿修持儀式。

### 4. 🚨 自動淡入淡出「正念警策提醒」
* **無感提醒，絕不中斷**：當持誦達到指定遍數（如 49 遍、108 遍）時，系統會自動在螢幕上方彈出警策字句。
* **無需滑鼠點擊確認**：提醒會在停留 5 秒後**全自動平滑淡出隱藏**。在此期間，**背景錄音與木魚共修音效完全不會暫停**，絕不打擾或中斷行者口誦心惟的持咒節奏。

### 5. 📊 終身功德累計大面板
* 包含「本次計數」與「終身功德總累計」，並以 10 萬遍為經典里程碑目標，直觀顯示功德進度條（如 10 萬遍得消除宿世惡業重障、百萬遍蒙菩薩現身加持），極大地增長修持信心與精進動力。
