📿 隨心修持助手 (Chanting & Mantra Assistant)
一個輕量、純前端的網頁版佛號與神咒隨身修持助手。專為需要流暢記錄、循環播放現場法音的修持者設計。本專案特別針對移動端（尤其是 iOS Safari）的音訊限制進行優化，確保在鎖屏或連續播放時能流暢無縫地循環。

🚀 項目已啟用 GitHub Pages 部署，即開即用。

🌟 核心功能
🎙️ 現場隨心錄音：只需按下錄音鍵，即可現場錄製一聲佛號或一句完整咒語（如：阿彌陀佛、準提神咒、心咒等）。

🔄 雙重持誦模式：

設定計數：內建 10 次、21 次、49 次、108 次及 1000 次等傳統功德圓滿計數，方便定課修持。

無限循環：無限制持續播放，適合靜坐、定時修持或環境佛號常持。

⚡ 毫秒級無縫重播：採用 HTML5 Audio 事件驅動優化，避開移動端常見的音訊延遲與卡頓，實現流暢的連播體驗。

📊 功德統計紀錄：利用瀏覽器 localStorage 本地儲存每次修持的日期、次數及是否圓滿，數據不外傳，完美保護個人隱私。

📑 內置工具與儀軌版面
本倉庫目前包含以下修持網頁：

1. 隨心現場錄音唸佛助手 (index.html)
簡潔版面，專為短佛號、聖號快速錄製連播設計。

快速計數與歷史功德榜。

2. 準提神咒修持助手 (zhunti.html 或覆蓋主頁)
配合正統準提法門儀軌設計，介面包含完整三段式結構：

① 開咒偈：內置「稽首皈依蘇悉帝，頭面頂禮七俱胝...」四句讚佛偈。

② 準提咒文：對齊呈現梵文、羅馬拼音、漢字譯音（全咒與心咒），方便邊聽邊對音。

③ 全列表功德迴向文：

移除繁瑣的下拉選單，一次過完整列出五大經典迴向範例（例一：準提鏡護家門、例二：消業增福、例三：求學業、例四：求人緣婚戀、例五：求財運）。

雙欄動態同步：只需在頂部輸入「信士姓名」與「祈福對象」，下方五個範例會即時自動帶入並高亮顯示。修持圓滿後，即可直接對著螢幕合掌讀誦，極致方便。

🛠️ 技術亮點
iOS 標準 Blob URL 模式：優先強制採用 audio/mp4 編碼錄製，自動釋放舊物件記憶體（revoking URL），徹底解決 iPhone 網頁錄音無法播放的通病。

繞過 Web Audio API Bug：棄用容易在 iOS 鎖屏或背景斷音的 Web Audio 節點控制，改用純 HTML5 原生事件監聽（ended 事件驅動），達成超低延遲的重播。

零後端依賴：無任何第三方 API 或伺服器交互，100% 本地安全運行。

📖 使用方法
打開網頁：進入部署好的網頁。

錄音：點擊 「開始錄製」，對著麥克風念誦一聲佛號或咒語，完成後再次點擊停止。

設定：選擇「設定計數」或「無限循環」，並挑選您的功德目標次數。

開始：點擊 「開始持誦」，系統將自動開始循環播放並即時累加計數。

迴向（準提頁面專屬）：在上方輸入您的姓名與祈福對象，下方的所有迴向文會即時更新。結課後，直接看著螢幕選讀對應的願文即可。

💡 iOS 使用小提示：因 Apple 安全機制限制，首次開始連播時必須由用戶主動點擊按鈕以觸發音訊。請確保手機未處於「靜音模式」，以免音訊被系統阻斷。
